###############
Troubleshooting
###############

This section talks about troubleshooting Yarn applications for the developer
team. Use the YARN CLI to view logs for running application. Configure the log
aggregation to aggregate and write out logs for all containers belonging to a
single Application grouped by NodeManagers to single log files at a configured
location in the file system.

To see a full list of Yarn Mapreduce/Yarn logs, refer to
:numref:`logging_monitoring_and_alerting_logs_location`.
